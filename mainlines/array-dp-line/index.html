数组DP线

从L3启动，贯穿到L8——数组思维与动态规划的长线培养。 这是时间跨度最长的能力主线。

数组DP线可能是最被低估的能力主线。 很多家长和学生觉得"数组很简单"，但真正的数组思维（下标意识、空间换时间、状态表示）需要长期培养。

GESP官方把前缀和放在L7，但我的实践表明：前缀和必须在L3与一维数组同步教。 晚教16周，就失去了培养数组思维的黄金窗口期。

为什么数组DP线必须从L3启动？

1. 下标思维的关键期

L3是建立"下标意识"的最佳时期：

  • 学生刚接触数组，还没有形成固定思维
  • 可以同时建立"数组+下标操作"的完整认知
  • 前缀和、桶计数等思想可以自然融入

如果等到L7再学前缀和，学生的数组思维已经固化，很难接受新的操作方式。

数组DP线的发展阶段

L3
数组初识期 启动
核心内容：
  • 一维数组基础操作
  • 下标遍历（正序、逆序）
  • 前缀和初步（同步教）
  • 桶计数（频率统计）
思维目标：建立"数组是下标到值的映射"的意识
L4
数组深化期 深化
核心内容：
  • 二维数组
  • 前缀和应用（区间求和）
  • 差分数组初步
  • 数组与函数结合
思维目标：理解"空间换时间"的思想
L5
递推入门期 过渡
核心内容：
  • 递推思想（斐波那契、阶乘）
  • 递推与递归的对比
  • 数组存储中间结果
思维目标：从递归走向DP的过渡
L6
一维DP期 分水岭
核心内容：
  • DP三要素：状态、转移、初值
  • 线性DP（最长递增子序列）
  • 背包问题初步
思维目标：建立"状态表示"的核心思维
L7
二维DP期 高级
核心内容：
  • 二维DP（最长公共子序列）
  • 区间DP
  • DP与图论结合
思维目标：多维状态表示
L8
DP优化期 精通
核心内容：
  • 状态压缩DP
  • 滚动数组优化
  • 单调队列优化
思维目标：空间与时间的平衡艺术

核心思维：从数组到DP

数组思维

下标是钥匙，值是宝藏。学会用下标操作数组。

关键技能 遍历、筛选、统计、前缀和

递推思维

当前值由前面的值推导而来。建立递推关系。

关键技能 找递推式、存储中间结果

DP思维

状态表示 + 状态转移。用数组存储最优解。

关键技能 设计状态、写转移方程

常见问题

Q1：前缀和为什么要在L3教？

前缀和的本质是"空间换时间"，用O(n)空间换取O(1)区间查询。这个思想越早建立越好。L3学生刚学数组，还没有形成"只存原始数据"的定式，容易接受"数组可以存衍生数据"的思维。

Q2：桶计数和数组有什么关系？

桶计数就是数组的经典应用：用下标表示数值，用值表示频率。这是"下标思维"的最佳训练。学生学会桶计数后，看到频率统计题就能秒想到数组。

Q3：DP和递归有什么关系？

DP和递归是同一枚硬币的两面。递归是"自顶向下"，DP是"自底向上"。L5-L6的阶段就是把递归思维转化为DP思维的过程。两条主线在这里交汇。
