1. LCA：最近公共祖先

LCA（Lowest Common Ancestor）是指树中两个节点的最深公共祖先。 它在树链查询、树上距离计算等问题中非常常用。

倍增法是求LCA最常用的方法：先通过DFS预处理每个节点的2^k级祖先， 然后让两个节点同时向上"跳"，直到相遇。

倍增LCA模板 
const int MAXN = 100005;
const int LOG = 20;

int depth[MAXN];
int parent[MAXN][LOG];
vector<int> tree[MAXN];

void dfs(int u, int p) {
    depth[u] = depth[p] + 1;
    parent[u][0] = p;
    for (int i = 1; i < LOG; i++) {
        parent[u][i] = parent[parent[u][i-1]][i-1];
    }
    for (int v : tree[u]) {
        if (v != p) dfs(v, u);
    }
}

int lca(int u, int v) {
    if (depth[u] < depth[v]) swap(u, v);
    // 先把u提升到和v同一深度
    for (int i = LOG - 1; i >= 0; i--) {
        if (depth[parent[u][i]] >= depth[v]) {
            u = parent[u][i];
        }
    }
    if (u == v) return u;
    // 同时向上跳
    for (int i = LOG - 1; i >= 0; i--) {
        if (parent[u][i] != parent[v][i]) {
            u = parent[u][i];
            v = parent[v][i];
        }
    }
    return parent[u][0];
}

预处理时间 O(n log n)，每次查询 O(log n)。

2. Tarjan强连通分量（SCC）

在有向图中，如果任意两个节点互相可达，这个子图就是一个强连通分量。 Tarjan算法可以在 O(n+m) 时间内求出所有的强连通分量。

Tarjan SCC模板 
int dfn[MAXN], low[MAXN], scc[MAXN];
bool inStack[MAXN];
stack<int> stk;
int timer = 0, sccCount = 0;

void tarjan(int u) {
    dfn[u] = low[u] = ++timer;
    stk.push(u);
    inStack[u] = true;
    
    for (int v : adj[u]) {
        if (!dfn[v]) {
            tarjan(v);
            low[u] = min(low[u], low[v]);
        } else if (inStack[v]) {
            low[u] = min(low[u], dfn[v]);
        }
    }
    
    if (dfn[u] == low[u]) {
        sccCount++;
        while (true) {
            int x = stk.top();
            stk.pop();
            inStack[x] = false;
            scc[x] = sccCount;
            if (x == u) break;
        }
    }
}

核心思想：dfn 是DFS序，low 是能回溯到的最小dfn。 当 dfn[u] == low[u] 时，说明u是一个强连通分量的"根"， 栈中从u到栈顶的所有节点构成一个SCC。

3. 缩点：把SCC变成DAG

缩点是Tarjan最重要的应用之一：把每个强连通分量缩成一个"超级节点"， 原图就变成了一个有向无环图（DAG）。

DAG上可以做拓扑排序、动态规划、最短路径等，很多问题因此变得可解。

缩点代码 
// 统计每个SCC的权值和
vector<long long> sccVal(sccCount + 1, 0);
for (int i = 1; i <= n; i++) {
    sccVal[scc[i]] += val[i];
}

// 构建DAG
vector<int> dag[MAXN];
for (int u = 1; u <= n; u++) {
    for (int v : adj[u]) {
        if (scc[u] != scc[v]) {
            dag[scc[u]].push_back(scc[v]);
        }
    }
}

缩点后，可以在DAG上做DP求"从某个SCC出发能到达的最大权值和"等经典问题。

4. 双连通分量（拓展）

在无向图中，边双连通分量是指删除任意一条边后仍然连通的子图。 割边（桥）则是指删除后会使图不连通的边。

求割边的方法与Tarjan求SCC非常相似，同样利用dfn和low数组： 如果 low[v] > dfn[u]，则边 u→v 是割边。

L8阶段要求：掌握割边的判定条件，理解双连通分量的概念即可。 复杂变体（点双连通分量）可在后续专项训练中深入学习。

IOI教练常见教学陷阱

陷阱1：LCA预处理时根节点深度设为0

根节点的父节点通常设为0，所以depth[0]应该设为0，depth[root] = 1。 否则深度比较时会出现边界错误。

陷阱2：Tarjan中inStack和visited混用

在Tarjan中，只有还在栈里的节点才能用来更新low值。 已经出栈的节点虽然被访问过，但不代表它和当前节点在同一个SCC中。

陷阱3：缩点后DAG有重边

缩点后，两个SCC之间可能有多条边。如果后续要做拓扑排序或DP， 重边一般不影响结果，但要注意不要重复计算。

LCA基础

强连通分量

重点

缩点DP

重点

给定有向图，每个节点有权值。求从某个节点出发，能到达的节点权值之和最大是多少。（提示：缩点+DAG DP）

割边

挑战

给定无向图，求所有的割边（桥）。
