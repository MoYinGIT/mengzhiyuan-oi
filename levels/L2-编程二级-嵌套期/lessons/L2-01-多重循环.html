L2-01 多重循环

6课时（270分钟） L2核心

教学目标

知识目标

  • 理解三重循环的结构和执行规律
  • 掌握内外层循环变量的独立控制
  • 理解多重循环的时间复杂度概念

技能目标

  • 能用三重循环打印复杂图形
  • 能控制各层循环的边界条件
  • 能计算多重循环的执行次数

从双重到三重循环

L1我们学了双重循环，L2要把这个能力再推进一步。

三重循环的本质和双重循环一样：外层执行一次，中层执行一轮；中层执行一次，内层执行一轮。

三重循环基本结构

C++ 
for (int i = 1; i <= n; i++) {        // 外层
    for (int j = 1; j <= n; j++) {    // 中层
        for (int k = 1; k <= n; k++) { // 内层
            // 执行语句
        }
    }
}

执行次数：如果三层都是 1 到 n，那么内层语句执行 n × n × n = n³ 次。

经典案例

案例1：打印数字金字塔

    1
   222
  33333
 4444444
555555555
C++ 
int n = 5;
for (int i = 1; i <= n; i++) {
    // 第一层：打印空格
    for (int j = 1; j <= n - i; j++) {
        cout << " ";
    }
    // 第二层：打印数字
    for (int k = 1; k <= 2 * i - 1; k++) {
        cout << i;
    }
    cout << endl;
}

案例2：空心正方形

*****
*   *
*   *
*   *
*****
C++ 
int n = 5;
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n; j++) {
        // 边界打印*，内部打印空格
        if (i == 1 || i == n || j == 1 || j == n) {
            cout << "*";
        } else {
            cout << " ";
        }
    }
    cout << endl;
}

IOI教练视角

陷阱：循环变量命名混乱

三重循环建议统一使用 i、j、k 作为循环变量名，不要混用其他名字。这是竞赛界的约定俗成，看到 i 就知道是最外层。

陷阱：边界条件写错

三重循环中，中层或内层的边界条件依赖外层变量时（如 j <= i），初学者容易写成固定值（如 j <= n），导致输出错误。

提高练习

  • 打印空心菱形
  • 输入 n，输出 n 阶数字螺旋矩阵
