L2-04 字符串基础

7课时（315分钟） L2核心

教学目标

知识目标

  • 理解字符串是字符的序列
  • 掌握 C 风格字符串的基本操作
  • 理解字符串结束符 '\0' 的作用

技能目标

  • 能声明、输入、输出字符串
  • 能计算字符串长度
  • 能遍历字符串并操作每个字符

字符串的本质：字符数组

字符串不是单一的数据类型，而是一个字符序列。

在 C++ 中，字符串以 char数组 的形式存储，末尾有一个特殊的结束符 '\0'，标记字符串的结束位置。

内存布局可视化

字符串声明与初始化

C++ 
#include <cstring>  // 使用字符串函数需要包含

// 方式1：直接初始化
char s1[] = "Hello";

// 方式2：指定大小（要比实际长度多1，留给'\0'）
char s2[20];

// 方式3：逐字符赋值（不推荐）
char s3[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

常用字符串操作

C++ 
char s[100];

// 1. 输入字符串
cin >> s;           // 读到空格停止
cin.getline(s, 100); // 读一整行，包括空格

// 2. 输出字符串
cout << s << endl;

// 3. 求字符串长度
int len = strlen(s);

// 4. 字符串复制
char t[100];
strcpy(t, s);       // 把 s 复制到 t

// 5. 字符串连接
strcat(s, " World"); // 把 " World" 接到 s 后面

// 6. 字符串比较
if (strcmp(s, "Hello") == 0) {
    cout << "相等" << endl;
}
重要陷阱：不能用 == 比较字符串

字符串比较必须用 strcmp()，不能用 ==。因为 == 比较的是地址，不是内容。这是初学者最常犯的错误之一。

字符串遍历

C++ 
char s[100];
cin >> s;

// 方式1：用 strlen
int len = strlen(s);
for (int i = 0; i < len; i++) {
    cout << s[i] << " ";
}

// 方式2：用结束符判断（更高效，不用计算长度）
for (int i = 0; s[i] != '\0'; i++) {
    cout << s[i] << " ";
}

// 应用：统计字符串中字母的个数
int count = 0;
for (int i = 0; s[i] != '\0'; i++) {
    if ((s[i] >= 'a' && s[i] <= 'z') || (s[i] >= 'A' && s[i] <= 'Z')) {
        count++;
    }
}

IOI教练视角

陷阱：数组越界

C 风格字符串没有边界检查，如果输入的字符串长度超过了数组声明的大小，会覆盖其他内存区域，导致程序崩溃或行为异常。

长线提示：L4会学 string 类，它比 char 数组更安全方便。但 char 数组是理解字符串底层原理的关键，必须先掌握 char 数组，再用 string 类时才能真正理解它在做什么。

练习设计

基础练习

  • 输入一个字符串，输出它的长度
  • 输入一个字符串，统计其中大写字母、小写字母、数字的个数
  • 输入一个字符串，将其中的所有大写字母转为小写

提高练习

  • 输入一个字符串，判断它是否是回文字符串
  • 输入两个字符串，判断第二个字符串是否是第一个的子串

自查清单

本课自查

  • 能正确声明和输入 C 风格字符串
  • 能用 strlen 求字符串长度
  • 能用循环遍历字符串的每个字符
  • 知道不能用 == 比较字符串
  • 如果没有完成以上检查项，不要进入下一课
