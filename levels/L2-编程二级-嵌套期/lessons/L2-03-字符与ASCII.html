L2-03 字符与ASCII

6课时（270分钟） L2核心

教学目标

知识目标

  • 理解字符的本质是整数（ASCII编码）
  • 掌握常见字符的ASCII码范围
  • 理解字符可以进行数学运算

技能目标

  • 能进行大小写转换
  • 能判断字符类型（字母/数字/空格）
  • 能利用ASCII码进行字符运算

字符的本质：披着字符外衣的数字

在计算机看来，'A' 和 65 没有本质区别。'A' 只是 65 的一种显示方式。

这就是 ASCII 编码的核心思想：每个字符都对应一个唯一的整数。

常见ASCII码

字符ASCII码说明
'0' ~ '9'48 ~ 57数字字符
'A' ~ 'Z'65 ~ 90大写字母
'a' ~ 'z'97 ~ 122小写字母
' '32空格
'\n'10换行符

内存布局可视化

字符运算

C++ 
// 大小写转换
char lower = 'a' - 32;    // lower = 'A'
char upper = 'A' + 32;    // upper = 'a'

// 数字字符转整数
char c = '5';
int num = c - '0';        // num = 5

// 整数转数字字符
int n = 7;
char ch = n + '0';        // ch = '7'

// 判断字符类型
char x;
cin >> x;
if (x >= 'A' && x <= 'Z') {
    cout << "大写字母" << endl;
} else if (x >= 'a' && x <= 'z') {
    cout << "小写字母" << endl;
} else if (x >= '0' && x <= '9') {
    cout << "数字" << endl;
}

IOI教练视角

陷阱：把数字字符当成整数

初学者常犯的错误：把 '5' 当成 5 参与运算。'5' 的 ASCII 值是 53，不是 5。要得到 5，必须做 '5' - '0'。

长线思维：L2学的字符运算是 L3 字符串处理、L4 字符数组、L6-L7 字符串算法的基础。这里不扎实，后面会处处卡壳。

练习设计

基础练习

  • 输入一个字符，如果是大写字母则转为小写，反之转为大写
  • 输入一个数字字符，输出它对应的整数值
  • 输出 'A' 到 'Z' 的所有字母

提高练习

  • 输入一个字符，判断它是字母、数字还是其他字符
  • 输入一个字母，输出它在字母表中的位置（A=1, B=2...）

自查清单

本课自查

  • 知道 'A'、'a'、'0' 的 ASCII 码
  • 能进行大小写转换
  • 能把数字字符和整数值相互转换
  • 能判断字符的类型
  • 如果没有完成以上检查项，不要进入下一课
