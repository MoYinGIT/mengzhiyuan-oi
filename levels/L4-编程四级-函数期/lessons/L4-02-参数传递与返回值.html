值传递、return语句、多返回值技巧

难度

1. 值传递：复制一份给函数

C++中，函数参数默认采用值传递：调用函数时，会把实参的值复制一份给形参。 函数内部对形参的修改，不会影响外部的实参。

值传递示例 
void addTen(int x) {
    x = x + 10;
    cout << "函数内 x = " << x << endl; // 输出 20
}

int main() {
    int a = 10;
    addTen(a);
    cout << "函数外 a = " << a << endl; // 输出 10（没变！）
    return 0;
}

上图说明：a 和 x 是两个独立的变量，只是值相同。 修改 x 不会影响 a。

2. 为什么值传递很重要？

值传递是C++的安全机制：函数内部的修改被"隔离"在函数内部，不会意外破坏外部数据。 但这也意味着，如果你希望函数修改外部变量，单纯传变量本身是不行的。

L4阶段的处理方式：如果要让函数"修改"外部变量，可以把修改后的值通过 return 返回， 然后在调用处重新赋值。引用传递（&）是更高级的内容，可在L5或L6再深入。

3. return 语句详解

return 的多种用法 
// 1. 返回计算结果
int square(int x) {
    return x * x;
}

// 2. 提前返回（常用于边界条件）
bool isPositive(int x) {
    if (x > 0) return true;
    return false;  // 等价于 else return false;
}

// 3. void 函数可以没有 return，也可以用 return 提前结束
void printIfPositive(int x) {
    if (x <= 0) return;  // 提前退出
    cout << x << endl;
}

注意：return 之后的代码不会执行。初学者常犯的错误是在 return 之后还写逻辑。

4. 多返回值的常用方案

C++函数只能直接返回一个值。如果需要返回多个结果，有几种常用方案：

方案1：用全局变量（L4阶段常用）

全局变量返回多值 
int maxVal, minVal; // 全局变量

void getMaxMin(int a[], int n) {
    maxVal = a[0];
    minVal = a[0];
    for (int i = 1; i < n; i++) {
        if (a[i] > maxVal) maxVal = a[i];
        if (a[i] < minVal) minVal = a[i];
    }
}

int main() {
    getMaxMin(arr, 5);
    cout << maxVal << " " << minVal << endl;
}

方案2：通过数组/结构体返回（更规范）

数组返回多值 
// 返回数组中最大值的【下标】和【值】
int getMaxIndex(int a[], int n) {
    int maxIdx = 0;
    for (int i = 1; i < n; i++) {
        if (a[i] > a[maxIdx]) maxIdx = i;
    }
    return maxIdx; // 返回下标，调用方用 a[maxIdx] 获取值
}

IOI教练常见教学陷阱

陷阱1：以为函数内修改参数会影响外部

这是L4最高频的Bug来源。要反复强调：值传递是"复制"，不是"共享"。 可以用内存图可视化讲解。

陷阱2：return 多个值

C++不支持 return a, b; 这种写法。 如果需要多返回值，现阶段可以用全局变量，或者返回下标/索引。

陷阱3：void函数写了return值

void f() { return 0; } 是编译错误。 void函数只能用 return; 或者直接不写return。
