L1-06 循环结构 - for

3课时（135分钟） L1核心

教学目标

知识目标

  • 理解循环的意义：重复执行代码
  • 掌握 for 循环的三要素语法
  • 理解循环变量的作用和变化规律

技能目标

  • 能用 for 循环实现固定次数的重复操作
  • 能实现累加、计数等常见循环应用
  • 能正确控制循环的起止条件

循环的本质：让计算机重复劳动

人类讨厌重复劳动，但计算机最擅长的就是重复劳动。

如果要打印1到100，你不需要写100行 cout。写一个循环，让计算机替你重复100次。

for 循环语法

C++ 
for (初始化; 循环条件; 更新操作) {
    // 循环体：每次循环执行的代码
}

for 循环三要素

初始化

设置循环变量的初始值。

示例 int i = 1;

循环条件

判断循环是否继续执行的条件。

示例 i <= 100

更新操作

每次循环结束后对循环变量的修改。

示例 i++

流程图可视化

flowchart TD A[初始化 i=1] --> B{i <= 100?} B -->|是| C[执行循环体] C --> D[i++] D --> B B -->|否| E[结束循环]

变量跟踪：累加过程

代码示例

C++ 
// 计算 1+2+3+...+100
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum = sum + i;
}
cout << "总和是：" << sum << endl;

// 打印 1 到 10
for (int i = 1; i <= 10; i++) {
    cout << i << " ";
}
cout << endl;

// 打印 n 个星号
int n;
cin >> n;
for (int i = 1; i <= n; i++) {
    cout << "*";
}

IOI教练视角

陷阱：边界错误

for 循环最容易犯的错误是边界控制。比如想循环10次，写了 i < 10（只有9次）或者 i <= 10（有11次）。

调试技巧：循环前先手动模拟 i 的取值，确认循环次数正确。

陷阱：循环变量在循环外使用

在 C++ 中，for (int i = 1; ...) 声明的 i 只在循环内部有效。初学者常在循环外使用 i 导致编译错误。

练习设计

基础练习

  • 用 for 循环计算 1 到 n 的和
  • 用 for 循环计算 n 的阶乘
  • 输入 n，输出 1 到 n 中所有偶数

提高练习

  • 输入 n，判断 n 是否为质数（只能被1和自身整除）
  • 输出 1 到 100 中所有能被3整除但不能被5整除的数
