L1-05 选择结构 - switch

3课时（135分钟） L1基础

教学目标

知识目标

  • 理解 switch 语句的适用场景
  • 掌握 switch-case 的语法结构
  • 理解 break 和 default 的作用

技能目标

  • 能使用 switch 处理多分支等值判断
  • 能正确在 case 后使用 break
  • 能区分 if-else if 和 switch 的使用场景

什么时候用 switch？

当你要判断的条件是一个变量的多个具体取值时，switch 比 if-else if 更清晰。

比如：输入 1-7 表示星期几，输出对应的星期名称。这种"等值多分支"最适合用 switch。

switch 语法结构

C++ 
switch (表达式) {
    case 值1:
        // 表达式等于值1时执行
        break;
    case 值2:
        // 表达式等于值2时执行
        break;
    case 值3:
        // 表达式等于值3时执行
        break;
    default:
        // 以上都不匹配时执行
        break;
}

完整示例：星期转换

C++ 
#include <iostream>
using namespace std;

int main() {
    int day;
    cin >> day;
    
    switch (day) {
        case 1:
            cout << "星期一" << endl;
            break;
        case 2:
            cout << "星期二" << endl;
            break;
        case 3:
            cout << "星期三" << endl;
            break;
        case 4:
            cout << "星期四" << endl;
            break;
        case 5:
            cout << "星期五" << endl;
            break;
        case 6:
            cout << "星期六" << endl;
            break;
        case 7:
            cout << "星期日" << endl;
            break;
        default:
            cout << "输入错误" << endl;
            break;
    }
    
    return 0;
}

break 的作用

核心陷阱：忘记 break

如果 case 后面没有 break，程序会继续执行下一个 case 的代码，这种现象叫"穿透"（fall-through）。

穿透示例

C++ 
switch (n) {
    case 1:
        cout << "一" << endl;
        // 没有 break！会穿透到 case 2
    case 2:
        cout << "二" << endl;
        break;
}

// 如果输入 1，输出：
// 一
// 二

初学建议：每个 case 后面都要写 break，不要故意利用穿透，除非你完全理解自己在做什么。

if-else if vs switch

对比项if-else ifswitch
适用条件范围判断、复杂条件等值判断
条件类型任意表达式整型、字符型
可读性分支多时代码冗长等值分支多时代码清晰
典型场景成绩等级、数值范围菜单选择、星期月份

练习设计

基础练习

  • 输入1-12的数字，输出对应的月份名称
  • 输入A/B/C/D，输出对应的成绩等级描述
  • 输入一个运算符（+ - * /），输入两个数，输出运算结果

提高练习

  • 输入年份和月份，输出该月有多少天（考虑闰年）

  • 能使用 default 处理非法输入
  • 能判断一个问题适合用 if 还是 switch
