致命陷阱：整数除法

在C++中，两个整数相除，结果仍然是整数，小数部分被直接截断（不是四舍五入）！

cout << 5 / 2 << endl;    // 输出 2
cout << 5.0 / 2 << endl;  // 输出 2.5（至少一个是double）

赋值运算符：= 不是等于

这是编程初学者最难以转变的思维：在数学中，a = 5 表示 a 等于 5；但在编程中，a = 5 表示把 5 放进变量 a 里。

更关键的是：在编程中，a = a + 1 不仅是对的，而且非常常用！它的意思是"把 a 原来的值加 1，再存回 a"。

变量跟踪可视化

复合赋值运算符

C++ 
int a = 10;

a += 5;   // 等价于 a = a + 5;   a 变为 15
a -= 3;   // 等价于 a = a - 3;   a 变为 12
a *= 2;   // 等价于 a = a * 2;   a 变为 24
a /= 4;   // 等价于 a = a / 4;   a 变为 6
a %= 4;   // 等价于 a = a % 4;   a 变为 2

运算优先级

优先级运算符结合性
1（最高）() 括号-
2* / %从左到右
3+ -从左到右
4= += -= 等从右到左

记忆口诀：先乘除后加减，有括号先算括号里。不确定的时候，多加括号总没错。

IOI教练视角

陷阱：整数除法导致计算错误

初学者做"平均数"题目时，经常写成 sum / n，结果因为整数除法而丢失小数部分。正确做法是至少将一个操作数转换为 double。

陷阱：混淆 = 和 ==

L2会学到 ==（判断是否相等），但初学者常常在选择结构里把 if (a == 5) 写成 if (a = 5)。这是典型的赋值当等于错误。

练习设计

基础练习

  • 计算并输出 17 / 5、17 % 5、17.0 / 5 的值，观察区别
  • 已知长方形的长和宽，计算周长和面积
  • 交换两个变量的值（不使用临时变量，提示：用加减法）

提高练习

  • 输入一个三位数，输出它的个位、十位、百位数字
  • 输入秒数，转换为小时:分钟:秒的格式

自查清单

本课自查

  • 能正确计算包含 + - * / % 的表达式
  • 理解整数除法和小数除法的区别
  • 知道 a = a + 1 的含义并会使用
  • 能使用 % 运算符提取数字的各位
  • 如果没有完成以上检查项，不要进入下一课
