L1-04 选择结构 - if

3课时（135分钟） L1基础

教学目标

知识目标

  • 理解条件判断的作用
  • 掌握 if、if-else、if-else if-else 的语法
  • 理解关系运算符和逻辑运算符

技能目标

  • 能根据条件选择不同的执行路径
  • 能正确使用 ==、!=、>、< 等关系运算符
  • 能组合多个条件进行判断

为什么需要选择结构？

现实生活中的很多问题，不是"一步一步做下去"就能解决的，而是需要根据条件做不同的选择。

比如：如果成绩大于等于60分，输出"及格"；否则，输出"不及格"。这就是选择结构。

if 语句流程图

flowchart TD A[开始] --> B{条件判断} B -->|条件成立| C[执行语句A] B -->|条件不成立| D[执行语句B] C --> E[结束] D --> E

三种 if 语句形式

1. 单分支 if

C++ 
if (条件) {
    // 条件成立时执行的代码
}

2. 双分支 if-else

C++ 
if (条件) {
    // 条件成立时执行
} else {
    // 条件不成立时执行
}

3. 多分支 if-else if

C++ 
if (条件1) {
    // 条件1成立
} else if (条件2) {
    // 条件2成立
} else if (条件3) {
    // 条件3成立
} else {
    // 以上都不成立
}

关系运算符与逻辑运算符

运算符含义示例结果
==等于5 == 5true
!=不等于5 != 3true
>大于5 > 3true
<小于5 < 3false
>=大于等于5 >= 5true
<=小于等于5 <= 3false
逻辑运算符含义示例
&&并且（AND）a > 0 && a < 100
||或者（OR）a < 0 || a > 100
!非（NOT）!(a == 0)

IOI教练视角

最致命错误：== 写成 =

初学者在选择结构中最容易犯的错误：把 if (a == 5) 写成 if (a = 5)。

后果：这不会报错，但会把 5 赋值给 a，然后条件永远为真。很多初学者因此被卡很久。

陷阱：边界条件

判断"成绩大于等于60分及格"，要写 >= 60，不能写 > 60。初学者常常漏掉等于的情况。

练习设计

基础练习

  • 输入一个整数，判断它是奇数还是偶数
  • 输入一个年份，判断它是否是闰年
  • 输入三个整数，输出其中的最大值

提高练习

  • 输入一个字符，判断它是大写字母、小写字母还是数字
  • 输入三角形的三条边，判断能否构成三角形，如果能，判断是什么类型的三角形

自查清单

本课自查

  • 能正确写出 if、if-else、if-else if 语句
  • 不会把 == 写成 =
  • 能使用 && 和 || 组合多个条件
  • 能处理边界条件（>=、<=）
  • 如果没有完成以上检查项，不要进入下一课
