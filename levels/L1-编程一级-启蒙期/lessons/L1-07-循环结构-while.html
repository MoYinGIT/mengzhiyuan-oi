L1-07 循环结构 - while

3课时（135分钟） L1核心

教学目标

知识目标

  • 理解 while 循环的适用场景
  • 掌握 while 和 do-while 的语法
  • 理解 while 和 for 的区别

技能目标

  • 能用 while 实现条件控制的循环
  • 能避免死循环
  • 能根据场景选择 for 或 while

while 循环：当条件满足时重复

for 循环适合"知道要循环多少次"的场景，而 while 循环适合"不知道要循环多少次，但知道什么时候停止"的场景。

比如：不断读取输入，直到输入为0时停止。这种场景用 while 最合适。

while 语法

C++ 
while (条件) {
    // 条件成立时执行
}

do-while 语法

C++ 
do {
    // 先执行一次，再判断条件
} while (条件);

代码示例

C++ 
// 示例1：不断读入数字，直到输入0停止
int n;
cin >> n;
while (n != 0) {
    cout << "你输入了：" << n << endl;
    cin >> n;  // 关键：必须更新条件变量，否则死循环！
}

// 示例2：求一个正整数的各位数字之和
int x;
cin >> x;
int sum = 0;
while (x > 0) {
    sum += x % 10;  // 取最后一位
    x /= 10;        // 去掉最后一位
}
cout << "各位数字之和：" << sum << endl;

// 示例3：do-while 至少执行一次
int password;
do {
    cout << "请输入密码：";
    cin >> password;
} while (password != 123456);
cout << "密码正确！" << endl;

for vs while：如何选择？

场景推荐原因
已知循环次数for三要素清晰，不易遗漏
未知循环次数，已知停止条件while条件控制更直观
至少需要执行一次do-while保证先执行再判断
遍历数组/序列for下标控制方便

IOI教练视角

最危险的陷阱：死循环

while 循环如果条件永远为真，就会陷入死循环，程序永远不会结束。初学者最容易犯的错误就是忘记在循环体内更新条件变量。

// 死循环示例！
int i = 1;
while (i <= 10) {
    cout << i << endl;
    // 糟糕！忘记写 i++ 了
}

练习设计

基础练习

  • 用 while 循环计算 1 到 n 的和
  • 用 while 循环求一个数的反转（如 123 -> 321）
  • 猜数字游戏：程序想一个1-100的数，用户不断猜测，直到猜对

提高练习

  • 输入两个正整数，用辗转相除法求最大公约数（GCD）
  • 用 while 循环判断一个数是否是回文数
