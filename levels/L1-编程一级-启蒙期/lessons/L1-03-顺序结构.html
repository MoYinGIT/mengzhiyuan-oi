L1-03 顺序结构

3课时（135分钟） L1基础

教学目标

知识目标

  • 理解程序按顺序执行的流程
  • 掌握 cin 和 cout 的基本用法
  • 理解输入、处理、输出的三段式结构

技能目标

  • 能独立完成包含输入、运算、输出的完整程序
  • 能使用 endl 和 \n 控制输出格式
  • 能同时读取多个输入数据

程序执行流程：输入 → 处理 → 输出

任何一个完整的程序，都可以概括为三个阶段：

  1. 输入（Input）：从外部获取数据
  2. 处理（Process）：对数据进行计算或操作
  3. 输出（Output）：将结果展示给用户

L1的核心目标，就是让学生能独立写出符合 IPO 模式的完整程序。

流程图可视化

flowchart LR A[开始] --> B[输入数据] B --> C[处理计算] C --> D[输出结果] D --> E[结束]

代码示例：计算长方形面积

C++ 
#include <iostream>
using namespace std;

int main() {
    // === 1. 输入 ===
    int length, width;
    cout << "请输入长和宽：";
    cin >> length >> width;
    
    // === 2. 处理 ===
    int area = length * width;
    
    // === 3. 输出 ===
    cout << "面积是：" << area << endl;
    
    return 0;
}

cin 与 cout 详解

cout 输出

C++ 
// 输出字符串
 cout << "Hello, World!" << endl;

// 输出变量
int age = 10;
cout << "年龄是：" << age << endl;

// 多个内容连续输出
cout << "长=" << length << ", 宽=" << width << endl;

// endl vs \n
cout << "第一行" << endl;   // 换行并刷新缓冲区
cout << "第二行\n";         // 仅换行，效率稍高

cin 输入

C++ 
// 读取一个变量
int n;
cin >> n;

// 连续读取多个变量（用空格或回车分隔）
int a, b;
cin >> a >> b;

// 读取小数
double x;
cin >> x;

// 读取字符
char c;
cin >> c;

IOI教练视角

陷阱：程序"卡住"了

初学者常犯的错误：程序运行后没有输出，看起来像卡住了。其实是因为程序在等待 cin 输入，但学生没有意识到需要输入数据。

解决方法：养成习惯——看到 cin，就要准备输入；写程序时，先输出提示信息（如"请输入："），再写 cin。

陷阱：输出格式问题

很多OJ题目对输出格式要求严格，多一个空格、少一个换行都可能导致 Wrong Answer。L1就要养成仔细核对输出格式的习惯。

练习设计

基础练习

  • 输入两个整数，输出它们的和、差、积、商
  • 输入圆的半径，计算并输出圆的周长和面积
  • 输入一个学生的语文、数学、英语成绩，输出总分和平均分

提高练习

  • 输入商品单价和数量，计算总价（考虑打折：满100打9折）
  • 输入一个四位数，输出它的各个数字之和

自查清单

本课自查

  • 能独立写出包含 cin 和 cout 的完整程序
  • 理解程序从上到下依次执行的顺序
  • 能同时读取多个输入数据
  • 输出格式正确，符合题目要求
  • 如果没有完成以上检查项，不要进入下一课
