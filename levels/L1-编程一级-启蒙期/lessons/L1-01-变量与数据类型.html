L1-01 变量与数据类型

3课时（135分钟） L1基础 入门级

教学目标

知识目标

  • 理解变量是存储数据的容器
  • 掌握三种基本数据类型：int、double、char
  • 理解变量必须先声明后使用

技能目标

  • 能正确声明和初始化变量
  • 能根据需求选择合适的数据类型
  • 能使用cout输出变量值

思维目标

  • 建立变量名是地址的别名的底层直觉
  • 理解数据类型的本质是存储空间和解释方式

核心概念：变量是存储数据的盒子

想象一下，你有一个盒子，你可以在盒子里放东西，给盒子贴标签，以后通过标签找到盒子里的东西。

变量就是这样一个盒子：变量名是标签，变量的值是盒子里的东西，变量的类型决定了盒子的大小和能放什么。

内存布局可视化

变量三要素

变量名

标识变量的名字，编程时通过名字访问变量。

命名规则 只能包含字母、数字、下划线，不能以数字开头

变量类型

决定变量能存储什么数据，占用多少内存。

三种基本类型 int整数、double小数、char字符

变量值

变量中实际存储的数据，可以被修改。

重要特性 变量的值可以被覆盖（赋值运算）

三种基本数据类型

类型 说明 示例 占用空间
int 整数类型 10, -5, 0, 10000 4字节
double 双精度浮点数（小数） 3.14, -0.5, 2.0 8字节
char 字符类型 A, b, 9, # 1字节

代码示例

C++ 
#include <iostream>
using namespace std;

int main() {
    // 1. 声明变量并初始化
    int age = 10;           // 整数类型
    double price = 3.5;     // 小数类型
    char grade = 'A';       // 字符类型
    
    // 2. 输出变量值
    cout << "年龄: " << age << endl;
    cout << "价格: " << price << endl;
    cout << "等级: " << grade << endl;
    
    return 0;
}

IOI教练视角：教学陷阱

陷阱1：变量未初始化就使用

初学者常常声明变量后忘记赋值就直接使用。未初始化的变量值是随机的，会导致程序行为不确定。

陷阱2：混淆整数除法和小数除法

在C++中，5 / 2 的结果是 2（整数除法），而不是 2.5。要得到 2.5，至少需要一个操作数是 double 类型。

陷阱3：字符和字符串混淆

'A' 是 char 类型，用单引号；"A" 是字符串类型，用双引号。初学者容易混用导致编译错误。

练习设计

基础练习

  • 声明三个变量分别存储你的年龄、身高（米）、姓名首字母，并输出
  • 交换两个int变量的值（引入临时变量）

提高练习

  • 计算并输出一个圆的周长和面积（半径为double类型）
  • 探究：int类型变量能存储的最大值是多少？

思考题

  • 为什么 char 用单引号，而 cout 输出字符串用双引号？

自查清单

本课自查

  • 能独立写出变量声明和初始化的代码
  • 能说出 int、double、char 的区别和适用场景
  • 知道变量命名的基本规则
  • 理解赋值运算的含义（不是数学等号）
  • 如果没有完成以上检查项，不要进入下一课
