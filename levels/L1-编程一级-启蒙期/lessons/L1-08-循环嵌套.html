L1-08 循环嵌套

3课时（135分钟） L1核心

教学目标

知识目标

  • 理解循环嵌套的概念和结构
  • 掌握双重 for 循环的执行规律
  • 理解内外层循环变量的关系

技能目标

  • 能用双重循环打印图形
  • 能用双重循环实现九九乘法表
  • 能计算双重循环的执行次数

循环嵌套的本质

外层循环执行一次，内层循环执行一轮。

如果外层循环执行 n 次，内层循环执行 m 次，那么内层循环体总共执行 n × m 次。

执行流程可视化

基本结构

C++ 
for (int i = 1; i <= 3; i++) {      // 外层循环
    for (int j = 1; j <= 3; j++) {  // 内层循环
        cout << "*";
    }
    cout << endl;  // 内层循环结束后换行
}

经典案例：九九乘法表

C++ 
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        cout << j << "*" << i << "=" << i * j << "\t";
    }
    cout << endl;
}

关键点：内层循环的终止条件是 j <= i，而不是 j <= 9。这决定了每一行输出的个数随行号递增。

IOI教练视角

陷阱：循环变量重名

初学者常在内外层循环中使用同一个变量名（如都用 i），导致内层循环修改了外层的循环变量，造成意外的行为。

陷阱：忘记换行

打印图形时，内层循环结束后一定要记得换行（cout << endl;），否则所有内容会打印在同一行。

练习设计

基础练习

  • 打印 n 行 n 列的矩形星号图
  • 打印九九乘法表
  • 打印直角三角形（每行星号数递增）

提高练习

  • 打印等腰三角形（带空格对齐）
  • 打印菱形
  • 打印空心正方形
