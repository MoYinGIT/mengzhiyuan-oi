1. 素数筛

素数筛用于快速找出 1~n 中的所有素数。最基础的是埃拉托斯特尼筛法（埃氏筛）。

埃氏筛 
const int MAXN = 1000005;
bool isPrime[MAXN];
vector<int> primes;

void sieve(int n) {
    for (int i = 2; i <= n; i++) isPrime[i] = true;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) {
            primes.push_back(i);
            for (int j = i * 2; j <= n; j += i) {
                isPrime[j] = false;
            }
        }
    }
}

埃氏筛的时间复杂度是 O(n log log n)。对于大部分竞赛场景已经足够。

2. 欧拉筛（线性筛）

欧拉筛是每个合数只被它的最小质因子筛一次，因此时间复杂度是严格的 O(n)。

欧拉筛 
void eulerSieve(int n) {
    for (int i = 2; i <= n; i++) isPrime[i] = true;
    for (int i = 2; i <= n; i++) {
        if (isPrime[i]) primes.push_back(i);
        for (int j = 0; j < primes.size() && i * primes[j] <= n; j++) {
            isPrime[i * primes[j]] = false;
            if (i % primes[j] == 0) break;  // 保证只被最小质因子筛到
        }
    }
}

L7阶段建议：埃氏筛必须熟练掌握；欧拉筛作为进阶内容了解。 大部分题目中埃氏筛已经够用，但追求极致效率时欧拉筛更优。

3. GCD与LCM

GCD（最大公约数）可以用辗转相除法（欧几里得算法）高效求解。

GCD与LCM 
// 辗转相除法
long long gcd(long long a, long long b) {
    return b == 0 ? a : gcd(b, a % b);
}

// LCM = 两数乘积 / GCD
long long lcm(long long a, long long b) {
    return a / gcd(a, b) * b;  // 先除再乘，防止溢出
}

C++17标准库中已经内置了 std::gcd（需要 include <numeric>）， 但竞赛中手动写一个更安全可靠。

4. 快速幂

快速幂用于高效计算 a^n mod p，时间复杂度 O(log n)。 它的核心思想是二分：把指数不断对半拆分。

快速幂模板 
long long fastPow(long long a, long long n, long long mod) {
    long long res = 1;
    a %= mod;
    while (n > 0) {
        if (n & 1) res = res * a % mod;
        a = a * a % mod;
        n >>= 1;
    }
    return res;
}

快速幂在数论、矩阵运算、组合计数中无处不在，是竞赛中的必会技能。

5. 模逆元入门

在模运算中，除法不像整数那样直接进行。 a / b mod p 通常要转化为 a × b⁻¹ mod p， 其中 b⁻¹ 叫做 b 在模 p 下的逆元。

当 p 是质数时，根据费马小定理： b⁻¹ ≡ b^(p-2) (mod p)

费马小定理求逆元 
// 当mod为质数时，求a在模mod下的逆元
long long modInverse(long long a, long long mod) {
    return fastPow(a, mod - 2, mod);
}

逆元是组合数学中"除法取模"的核心工具，也是后续学习组合数、概率DP的基础。

IOI教练常见教学陷阱

陷阱1：筛法数组开太小或类型错误

筛法需要开到n+5的大小。如果n是10^6，数组必须开到1000005以上。 另外isPrime应该用bool或vector<bool>，不要用int浪费空间。

陷阱2：LCM先乘后除导致溢出

计算LCM时应该 a / gcd(a,b) * b，而不是 a * b / gcd。 否则a*b可能先溢出。

陷阱3：模逆元的前提条件不满足

费马小定理求逆元要求模数是质数，且a与模数互质。 如果模数不是质数，需要用扩展欧几里得算法。
