树形DP

树上最大独立集、树的最长路径、依赖背包、换根DP入门

10课时
约2.5周
难度

本课三维目标

知识 树形DP状态定义、DFS后序遍历、子树合并、树上前缀和
技能 能用树形DP解决最大独立集、树直径、依赖背包等经典问题
思维 建立"子树即子问题，递归返回子树答案"的树形DP信念

1. 什么是树形DP？

树形DP就是在树结构上进行的动态规划。 树有一个天然的优势：以任意节点为根，它的每个子树都是独立的子问题。 因此树形DP通常配合DFS后序遍历进行：先处理完所有子节点，再处理当前节点。

核心思想：当前节点的状态，由它的各个子节点的状态转移而来。

2. 经典问题：树上最大独立集

问题：给定一棵树，每个节点有一个权值。选出一些节点，使得任意两个被选中的节点不相邻， 求选中节点的最大权值和。

状态定义：

  • dp[u][0]：不选节点u时，以u为根的子树能获得的最大权值
  • dp[u][1]：选节点u时，以u为根的子树能获得的最大权值
树上最大独立集 
int dp[MAXN][2];
int value[MAXN];
vector<int> tree[MAXN];

void dfs(int u, int parent) {
    dp[u][0] = 0;
    dp[u][1] = value[u];
    
    for (int v : tree[u]) {
        if (v == parent) continue;
        dfs(v, u);
        
        // 如果不选u，子节点可选可不选
        dp[u][0] += max(dp[v][0], dp[v][1]);
        
        // 如果选u，子节点一定不能选
        dp[u][1] += dp[v][0];
    }
}

答案就是 max(dp[root][0], dp[root][1])。

3. 树的直径（最长路径）

问题：求树中任意两点之间最长路径的长度。

树形DP解法：对每个节点u，记录以u为端点向下走的最长路径 d1 和次长路径 d2。 经过u的最长路径就是 d1 + d2。全局最优即树的直径。

树直径（树形DP版） 
int ans = 0;

int dfs(int u, int parent) {
    int d1 = 0, d2 = 0;  // 最长和次长向下路径
    
    for (auto [v, w] : tree[u]) {
        if (v == parent) continue;
        int d = dfs(v, u) + w;
        
        if (d >= d1) {
            d2 = d1;
            d1 = d;
        } else if (d > d2) {
            d2 = d;
        }
    }
    
    ans = max(ans, d1 + d2);
    return d1;
}

这个方法比两遍BFS更通用，尤其适用于带权树和需要统计直径数量的问题。

4. 依赖背包（树上背包）

问题：树中每个节点是一个物品，选一个节点必须先选它的父节点。 在总容量限制下，求最大价值。这就是依赖背包或树上背包。

树上背包模板 
// dp[u][j] = 以u为根的子树中，花费为j时的最大价值
int dp[MAXN][MAXV];
int cost[MAXN], val[MAXN];

void dfs(int u) {
    // 先预留选u本身的费用
    for (int j = cost[u]; j <= V; j++) {
        dp[u][j] = val[u];
    }
    
    for (int v : tree[u]) {
        dfs(v);
        // 把子树v合并到u上（类似0/1背包，倒序）
        for (int j = V; j >= cost[u]; j--) {
            for (int k = 0; k <= j - cost[u]; k++) {
                dp[u][j] = max(dp[u][j], dp[u][j-k] + dp[v][k]);
            }
        }
    }
}

核心是把每个子树看作一个"物品组"，用背包的方式合并到父节点上。 注意容量要倒序遍历，避免子树内的物品被重复计算。

5. 换根DP（拓展）

有些问题要求以每个节点为根时的答案，比如"以每个节点为根的子树深度之和"。 如果对每个节点单独做一次DFS，复杂度是 O(n²)。

换根DP的思想：先做一遍DFS求出以某个固定根（如1）的答案， 然后通过状态转移，把根从父节点"换"到子节点，递推求出所有节点的答案。

L7阶段要求：理解换根DP的基本思想即可，不需要掌握复杂变体。 重点还是树上最大独立集和树直径。

IOI教练常见教学陷阱

陷阱1：忘记记录父节点导致死循环

树是无向图，DFS时如果不跳过父节点，会在两个节点之间来回走，造成死循环。 每个树形DP的DFS都必须带parent参数。

陷阱2：树的存储方式错误

树的边是无向的，读入时要双向添加。很多学生只加了一条边，导致子树遍历不完整。

陷阱3：树上背包复杂度爆炸

朴素树上背包是 O(nV²)，在n和V都很大时会TLE。 L7阶段先掌握基本写法，优化版本在L8再深入学习。
