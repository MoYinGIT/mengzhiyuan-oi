代码模板

常用代码模板、快读模板、标准框架

重要提醒

模板是工具，不是拐杖。学习初期不要依赖模板，要理解每一行代码的含义。 模板的价值在于减少重复劳动，而不是代替思考。

}

快读模板

C++ 
// 快速读入
inline int read() {
    int x = 0, f = 1;
    char ch = getchar();
    while (ch < '0' || ch > '9') {
        if (ch == '-') f = -1;
        ch = getchar();
    }
    while (ch >= '0' && ch <= '9') {
        x = x * 10 + ch - '0';
        ch = getchar();
    }
    return x * f;
}

DFS模板

C++ 
const int MAXN = 1005;
int n, m;
bool vis[MAXN][MAXN];
int dx[] = {0, 0, 1, -1};
int dy[] = {1, -1, 0, 0};

void dfs(int x, int y) {
    // 边界检查
    if (x < 1 || x > n || y < 1 || y > m) return;
    // 访问检查
    if (vis[x][y]) return;
    
    vis[x][y] = true;
    
    // 四个方向搜索
    for (int i = 0; i < 4; i++) {
        int nx = x + dx[i];
        int ny = y + dy[i];
        dfs(nx, ny);
    }
}
