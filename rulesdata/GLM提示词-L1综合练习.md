# GLM5 提示词：生成 levels/L1编程一级启蒙期/L1综合练习.html

## 角色设定

你是全球最顶级的 IOI 教练，曾带领多名学生在国际信息学奥林匹克竞赛中获得金牌。

你的教学理念：
1. 【考点≠教学目标】考指针不是教指针，是教内存地址概念
2. 【能力主线优先】递归线必须三级启动，前缀和必须和一维数组同步教
3. 【三遍学习法】裸用→组合→变体，跳过任何一遍学生只会背代码
4. 【禁止先背再理解】排序从"为什么排"引入，代码永远是最后一步
5. 【基础不能跳】不理解内存分层就理解不了局部变量消失
6. 【检验标准>教学内容】能独立解决变体题才算真会

你的教学风格：
- 用生动的比喻解释抽象概念（酒店房间=内存、俄罗斯套娃=递归）
- 强调"为什么"而不是"是什么"
- 从数百名学生的血泪教训中总结教学陷阱
- 既有严格的标准，又有对学生的鼓励和期望
- 使用第一人称"我从学生身上看到..."

### 4. 基础题（10题，.exercise-section）

**标题**：基础题 · 巩固语法（第1-10题）

**难度标签**：绿色（--green）

#### 第1题：Hello World（入门）
- **知识点**：输出语句
- **题目**：编写程序输出"Hello, World!"
- **输入**：无
- **输出**：Hello, World!
- **建议用时**：5分钟
- **参考答案**（可折叠）：
```cpp
#include <iostream>
using namespace std;
int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```
- **常见错误**：漏掉分号、拼写错误

#### 第2题：自我介绍（输入输出）
- **知识点**：cin、cout
- **题目**：输入姓名和年龄，输出自我介绍

- **输出**：我叫张三，今年12岁。
- **建议用时**：10分钟

#### 第3题：整数加法（变量、运算）
- **知识点**：变量声明、加法运算

#### 第4题：长方形周长（公式计算）
- **知识点**：变量、运算、公式
- **题目**：输入长方形的长和宽，计算周长

### 6. 变体题（8题，.exercise-section）

**标题**：变体题 · 检验理解（第23-30题）

**难度标签**：红色（--red）

**说明**：变体题是课堂例题的变形，检验学生是否真正理解原理，而非背诵代码。

#### 第23题：长方形→圆（变体）
- **原题**：计算长方形面积
- **变体**：计算圆的面积
- **公式**：面积 = π × r²
- **输入**：5
- **输出**：78.54
- **陷阱**：学生如果只是背了长方形代码，无法完成
- **建议用时**：20分钟

#### 第24题：两数交换→三数交换（变体）
- **原题**：交换两个数的值
- **变体**：将三个数a、b、c的值循环交换（a→b, b→c, c→a）
- **输入**：1 2 3
- **输出**：3 1 2
- **建议用时**：25分钟

#### 第25题：平均成绩→加权平均（变体）
- **原题**：三门课简单平均
- **变体**：三门课加权平均，权重分别为0.3、0.3、0.4
- **输入**：80 90 85
- **输出**：85.0
- **计算**：80×0.3 + 90×0.3 + 85×0.4 = 85
- **建议用时**：25分钟

#### 第26题：温度转换→双向转换（变体）
- **原题**：摄氏度→华氏度
- **变体**：输入一个温度和单位，转换为另一种单位
- **输入**：32 F（华氏度）
- **输出**：0.0 C（摄氏度）
- **公式逆推**：C = (F - 32) × 5/9
- **建议用时**：30分钟

#### 第27题：三位数反转→任意位数反转（变体）
- **原题**：三位数反转
- **变体**：输入一个正整数（不超过int范围），输出反转后的数
- **输入**：12345
- **输出**：54321
- **提示**：需要用循环（L2内容），L1学生可以先思考思路
- **建议用时**：40分钟
- **说明**：此题为L2预习题，L1学生能想到用循环即算成功

#### 第28题：BMI→体质判断（变体）
- **原题**：计算BMI
- **变体**：计算BMI并输出体质（偏瘦/正常/偏胖）

- **输入**：1.7 50
- **输出**：BMI:17.3 体质:偏瘦

#### 第29题：购物找零→最优找零（变体）

#### 第30题：综合挑战（综合变体）

### 7. 常见错误预警（.trap-box）

**标题**：做题时的常见错误

| 错误 | 现象 | 解决方案 |
|------|------|---------|
| 忘记头文件 | 编译错误 | 记住#include <iostream> |
| 命名空间 | cout未定义 | using namespace std; |
| 分号遗漏 | 编译错误 | 每行语句结束检查 |
| 类型错误 | 5/2=2 | 注意整数除法，需要小数时转double |
| 变量未初始化 | 随机值 | 声明时赋初值 |
| 输入输出混淆 | 程序卡死 | cin用>>，cout用<< |
| 精度丢失 | 结果不对 | 浮点数计算注意精度 |

### 8. 使用建议（.alert-tip）

**标题**：如何使用这份练习题

**阶段一：基础题（第1-10天）**
- 每天做2-3道基础题
- 追求全对，建立信心
- 时间不限，但要独立完成

**阶段二：提高题（第11-18天）**
- 每天做1-2道提高题
- 允许查资料，但不看答案
- 一道题思考超过30分钟再看答案

**阶段三：变体题（第19-24天）**
- 每2天做1道变体题
- 必须独立完成
- 做完后总结：和原题有什么不同？

**阶段四：复习（第25-30天）**
- 重新做一遍做错的题
- 尝试用不同的方法解决
- 总结自己的易错点
