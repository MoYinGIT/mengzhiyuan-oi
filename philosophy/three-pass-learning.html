铁律三详解

三遍学习法

裸用→组合→变体。每个概念必须经历的三遍学习过程。跳过任何一遍，学生只会背代码，不会用思想。

我观察了数百名学生的学习轨迹，发现了一个规律：真正掌握一个概念的学生，都经历了三个阶段—— 第一阶段孤立使用，第二阶段组合应用，第三阶段灵活变体。 缺少任何一个阶段，掌握都是不牢固的。

三遍学习详解

裸用（孤立使用）

单独使用新概念，理解其基本用法和原理。不要与其他复杂概念混合，确保学生掌握核心机制。

教学目标
  • 理解概念的基本语法
  • 能独立完成基础练习
  • 建立初步的直觉
for循环示例 写简单的累加、打印图形、遍历数组。只练循环本身，不加if判断。

组合（结合应用）

将新概念与已学知识结合，理解概念间的协作关系。这是从孤立知识到知识网络的过渡。

教学目标
  • 能与其他概念结合使用
  • 理解概念间的依赖关系
  • 解决综合题目
for循环示例 for循环+if判断，实现筛选、统计、找最大最小值。

变体（灵活运用）

解决变形题目，理解概念的本质，而非表面形式。能独立应对变体题，才算真正掌握。

教学目标
  • 能处理边界变化
  • 能应对条件变形
  • 真正理解本质
for循环示例 变形循环边界、倒序遍历、步长不为1、多重循环嵌套。

为什么需要三遍？

只走第一遍的问题

学生只能照葫芦画瓢，题目稍有变化就不会。他们记住了代码的形式，但没理解代码的本质。 考试时遇到变体题，只能凭运气蒙。

典型表现

"老师，这个题和例题长得不一样，我不会做..."

走完三遍的效果

学生理解概念的本质，能灵活应对各种变体。他们知道"为什么要这样做"， 而不仅仅是"怎么做"。遇到新题目能举一反三。

"这个题虽然是新的，但核心思想和那道题一样..."

实例：for循环的三遍学习

第一遍：裸用（2课时）

C++ 
// 基础练习1：打印1到10
for (int i = 1; i <= 10; i++) {
    cout << i << endl;
}

// 基础练习2：计算1到100的和
int sum = 0;
for (int i = 1; i <= 100; i++) {
    sum = sum + i;
}
cout << sum << endl;

// 基础练习3：打印n个星号
int n;
cin >> n;
for (int i = 1; i <= n; i++) {
    cout << "*";
}

第一遍要点：只练循环本身，不涉及条件判断、数组等其他概念。目标是建立"循环就是重复执行"的基本直觉。

第二遍：组合（2课时）

C++ 
// 组合练习1：统计奇数的个数
int count = 0;
for (int i = 1; i <= n; i++) {
    if (i % 2 == 1) {  // 结合if判断
        count++;
    }
}

// 组合练习2：找最大值
int maxVal = -1;
for (int i = 1; i <= n; i++) {
    int x;
    cin >> x;
    if (x > maxVal) {  // 结合变量比较
        maxVal = x;
    }
}

// 组合练习3：累加符合条件的数
int sum = 0;
for (int i = 1; i <= n; i++) {
    if (i % 3 == 0 || i % 5 == 0) {  // 结合逻辑运算
        sum += i;
    }
}

第二遍要点：for循环与if判断、变量运算结合。理解"循环是框架，内部可以嵌套各种逻辑"。

第三遍：变体（2课时）

C++ 
// 变体练习1：倒序遍历（边界变化）
for (int i = n; i >= 1; i--) {
    cout << i << endl;
}

// 变体练习2：步长不为1（条件变形）
for (int i = 2; i <= n; i += 2) {  // 只遍历偶数
    cout << i << endl;
}

// 变体练习3：双重循环嵌套（本质理解）
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++) {  // 内层循环边界依赖外层
        cout << "*";
    }
    cout << endl;
}

// 变体练习4：while改写（形式变化，本质不变）
int i = 1;
while (i <= n) {
    cout << i << endl;
    i++;
}

第三遍要点：各种变形：倒序、步长变化、嵌套、改用while。目标是理解"循环的本质是控制流程"，而非特定的语法形式。

实施建议

给老师

  • 每概念至少分配3个课时走三遍（裸用1课时+组合1课时+变体1课时）
  • 变体题必须占练习题量的50%以上
  • 如果学生在变体环节卡壳，说明前两遍没走好，需要回头补
  • 不要为了赶进度跳过第三遍——这是最容易被跳、却最重要的一遍
  • 宁可少讲几个概念，也要把每个概念走通三遍

给学生/家长

  • 学习时主动寻找变体题练习，不要只做和例题一样的题目
  • 做完题后问自己：如果条件变了，我还能做吗？
  • 遇到变体题卡壳是好事，说明发现了知识漏洞
  • 不要满足于"能做原题"，要追求"理解本质"
  • 如果只能背代码而不能灵活应变，说明还需要继续练习
